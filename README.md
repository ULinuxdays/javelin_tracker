# Javelin Tracker

## Purpose
Placeholder for documenting project decisions and usage instructions as the tracker evolves.

## Project Layout
- `javelin_tracker/`: Core package modules and CLI entry points
- `tests/`: Test suite
- `data/`: Local datasets (ignored from version control)

## Usage
- Activate the virtual environment: `source .venv/bin/activate`
- Extend `cli.py` with command definitions as features are implemented
- Add persistent storage helpers in `storage.py` and analysis routines in `analysis.py`
